Class: ChemStructureObjectGroup

Kekule. ChemStructureObjectGroup

new ChemStructureObjectGroup(id)

A group of structure object. Each items in the group is in attrib=>obj form.
Parameters:
Name Type Description
id String Id of this group.
Properties:
Name Type Description
items Array Items in this group. Each item is a hash: {obj, amount, role...} where obj is the real ChemStructureObject and the rests are related attributes.
raiseExceptionOnTypeMismatch Bool Whether raise exception when wrong type of object is added to group.
