Class: ContentBlock

ContentBlock

new ContentBlock(id, coord2D, coord3D)

Represent an block of content in chem document (especially in 2D renderingï¼Œdrawing in rect), like text block or image.
Parameters:
Name Type Description
id String Id of this block.
coord2D Hash The top left 2D coordinates of block, {x, y}, can be null.
coord3D Hash The min 3D coordinates of block box, {x, y, z}, can be null. Usually unneeded.
Properties:
Name Type Description
coord2D Hash The 2D coordinates of glyph, {x, y}.
coord3D Hash The 3D coordinates of glyph, {x, y, z}.
absCoord2D Hash The absolute 2D coordinates of glyph, {x, y}.
absCoord3D Hash The absolute 3D coordinates of glyph, {x, y, z}.
size2D Hash The 2D dimension (width, height) of glyph, {x, y}.
size3D Hash The 3D dimension (width, height, depth) of glyph, {x, y, z}.
