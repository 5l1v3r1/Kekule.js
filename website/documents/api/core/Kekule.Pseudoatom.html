Class: Pseudoatom

Kekule. Pseudoatom

new Pseudoatom(id, atomType, coord2D, coord3D)

Represent an unreal atom, such as dummy atom (A point or object with no chemical semantics), a "any atom" and so on. Examples can be centroids, bond-midpoints, orienting "atoms" in small z-matrices.
Parameters:
Name Type Description
id String Id of this node.
atomType String Unreal atom type, value from Kekule.PseudoatomType.
coord2D Object The 2D coordinates of node, {x, y}, can be null.
coord3D Object The 3D coordinates of node, {x, y, z}, can be null.
Properties:
Name Type Description
atomType String Unreal atom type, value from Kekule.PseudoatomType.
symbol String User custom symbol. Meanless if atomType != Kekule.PseudoatomType.CUSTOM.
