Class: StructureConnectionTable

Kekule. StructureConnectionTable

new StructureConnectionTable(owner, parent)

A connection table for representing a complex (especially organic) molecule or group.
Parameters:
Name Type Description
owner Kekule.ChemSpace Owner for each objects in connection table.
parent Kekule.StructureFragment Parent to hold this Ctab.
Properties:
Name Type Description
owner Kekule.ChemSpace Owner for each objects in connection table.
parent Kekule.StructureFragment Parent to hold this Ctab. Read only.
nodes Array All structure nodes in this connection table.
anchorNodes Array Nodes that can have bond connected to other structure nodes.
connectors Array Connectors (usually bonds) in this connection table.
nonHydrogenNodes Array All structure nodes except hydrogen atoms in this connection table.
nonHydrogenConnectors Array Connectors except ones connected to hydrogen atoms in this connection table.
