Class: Reaction

Kekule. Reaction

new Reaction()

A chemical reaction. Including information on reactants, products, substances and conditions. All informations are organized in a role-object form using Kekule.RoleMap. Property reactants, products and substances should be maps of role-Kekule.Molecule. Property conditions can be of role-Object.
Properties:
Name Type Description
name String Name of reaction.
title String Title of reaction.
reactionType String Type of reaction.
yield Float Yield of reaction. 0 <= Value <= 1.
reactants Array Reactants of reaction, each item in array should be a role map based hash.
products Array Products of reaction, each item in array should be a role map based hash.
substances Array Substances related but not evolved in reaction, such as solvent. Each item in array should be a a role map based hash.
conditions Array Conditions of reaction, including temperature, pressure and so on.
